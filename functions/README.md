* **Formatting Output**
  * Println
  * Printf - format value
  * Sprintf - returns a string instead of printing it
    |Verb  |Output  |
    |---------|---------|
    | %f     | floating point number         |
    | %d     | decimal integer         |
    | %s     | String         |
    | %t     | Boolean        |
    | %v     | any value        |
    | %#v     | any value as formatted in Go language        |
    | %T     | Type of supplied value (int,string,etc)       |
    | %%     | A literal percent sign         |
